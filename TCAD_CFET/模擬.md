## 1. 元件結構定義

### 1.1 GAAFET（Nanosheet）幾何參數

| 參數 | 典型值（3nm 節點） |
|------|-------------------|
| Nanosheet 寬度 (Wns) | 15–30 nm |
| Nanosheet 厚度 (Tns) | 4–6 nm |
| Nanosheet 間距 (Spacing) | 8–12 nm |
| Gate length (Lg) | 12–18 nm |
| EOT (等效氧化層厚度) | 0.5–0.7 nm |
| Inner spacer 長度 | 4–6 nm |

### 1.2 CFET 堆疊結構

CFET 將 nFET 與 pFET 垂直堆疊，底層通常為 nFET，頂層為 pFET（或相反，依製程而定）。

```
  ┌─────────────────────┐  ← Top Gate (pFET)
  │  pFET Nanosheet ×N  │
  ├─────────────────────┤  ← Middle Dielectric Isolation
  │  nFET Nanosheet ×N  │
  └─────────────────────┘  ← Bottom Gate (nFET)
```

關鍵尺寸：
- **中間介電隔離層厚度**：≥ 10 nm（避免靜電耦合）
- **獨立 Gate contact**：上下層需分開引線，在 SDE 中須明確定義 contact region

### 1.3 SDE 建構建議

```scheme
; 建議使用 Sentaurus Structure Editor (SDE) 腳本化建構
; 避免 GUI 手動操作，確保可重現性

(sdegeo:create-cuboid
  (position x0 y0 z0) (position x1 y1 z1)
  "Silicon" "nanosheet_1")

; Inner spacer 使用 SiOCN 或 SiN
(sdegeo:create-cuboid ... "SiOCN" "inner_spacer")

; High-k 使用 HfO2
(sdegeo:create-cuboid ... "HfO2" "gate_dielectric")
```

## 2. 網格策略

次3nm 元件的網格品質直接決定收斂性與精度。

### 2.1 關鍵區域網格密度

| 區域 | 最大網格尺寸 | 原因 |
|------|-------------|------|
| Channel（Nanosheet 厚度方向） | 0.3–0.5 nm | 量子限制效應需解析波函數 |
| Gate dielectric / 介面 | 0.1–0.2 nm | 陷阱、EOT 精度 |
| Source/Drain extension | 0.5–1.0 nm | 短通道效應、DIBL |
| Inner spacer 附近 | 0.5 nm | 寄生電容、電場集中 |
| Bulk / STI | 5–10 nm | 相對不關鍵，節省計算量 |

### 2.2 網格細化指令（SDE）

```scheme
(sdedr:define-refinement-size "ref_channel"
  (/ Tns 10) (/ Tns 10) (/ Tns 10)   ; x y z 方向
  (/ Tns 20) (/ Tns 20) (/ Tns 20))  ; 最小尺寸

(sdedr:define-refinement-region "ref_channel"
  "channel_region" "ref_channel")

; 介面細化（高梯度區域）
(sdedr:define-refinement-function "ref_interface"
  "DopingConcentration" "MaxTransDiff" 1)
```

### 2.3 注意事項

- Nanosheet 厚度方向至少需要 **10 個節點以上**，否則量子限制模型（如 density gradient）精度不足
- 避免網格長寬比（aspect ratio）超過 **50:1**，否則矩陣條件數惡化，導致求解發散
- CFET 雙層結構的中間隔離層網格不可過疏，建議至少 5 個節點

## 3. 物理模型選擇

### 3.1 必要物理模型

```
Physics {
  Mobility(
    DopingDep                    # 摻雜依賴遷移率（Masetti model）
    Enormal(                     # 垂直電場遷移率退化（Lombardi/Pregaldo）
      Lombardi
    )
    HighFieldSaturation(         # 高場速度飽和（van Overstraeten）
      GradQuasiFermi
    )
  )

  Recombination(
    SRH(
      DopingDep                  # 摻雜依賴 SRH 壽命
      TempDependence
    )
    Auger                        # 高摻雜區必要（S/D extension）
    Band2Band(                   # 帶帶穿隧（BTB）：BTBT 洩漏電流不可忽略
      E1                         # Klaassen BTB model
    )
  )

  QuantumPotential               # Density Gradient 量子修正（必要）

  Fermi                          # 使用 Fermi-Dirac 統計（高摻雜區精度）
}
```

### 3.2 量子效應模型選擇說明

次3nm 通道的量子限制效應不可用 Boltzmann 近似忽略：

| 模型 | 適用情境 | 計算成本 |
|------|---------|---------|
| **Density Gradient (DG)** | 通用，收斂性佳，適合工程模擬 | 低–中 |
| **Modified Local Density Approx. (MLDA)** | 介面反型層電荷分佈精度較高 | 中 |
| **Schrödinger-Poisson** | 最精確，適合學術研究 | 極高 |

**建議**：工程模擬用 `QuantumPotential`（DG），學術精度需求用 `eQuantumPotential + hQuantumPotential`（MLDA）。

### 3.3 能帶模型

```
Physics {
  BandGapNarrowing(OldSlotboom)  # 高摻雜能帶縮窄
  Bandstructure(                 # 應變通道能帶修正（如有 SiGe）
    Strain
  )
}
```

- SiGe channel（p-type）：必須啟用 `Strain` + `Bandstructure`，否則遷移率嚴重低估
- Si channel（n-type）：`EffectiveIntrinsicDensity(BandGapNarrowing(OldSlotboom))` 即可

### 3.4 Metal Gate / High-k 介面模型

```
Physics(MaterialInterface="HfO2/Silicon") {
  Traps(
    FixedCharge                  # 固定介面電荷（影響 Vth）
    eTrappedCharge               # 可捕獲電荷（動態特性）
  )
}
```

## 4. 數學求解器設定

### 4.1 基本求解器配置

```
Math {
  Extrapolate                    # 偏壓步進時外插初始猜測，加速收斂
  Derivatives                    # 計算雅可比矩陣時包含所有偏導數
  RelErrControl                  # 使用相對誤差控制（比絕對誤差更穩健）

  Iterations = 20                # Newton 最大迭代次數（預設 15，複雜結構可調高）
  NotDamped = 50                 # 不阻尼迭代次數上限

  Digits = 5                     # 收斂精度（10^-5），平衡精度與速度
  ErRef(electron) = 1e8          # 電子參考濃度（避免低載子區偽收斂）
  ErRef(hole)     = 1e8

  DirectCurrentComputation       # 直接電流計算（避免數值噪音）

  Method = ILS                   # 迭代線性求解器（大型 3D 結構推薦）
  # Method = ParDiSo             # 直接求解器（小型結構或 2D 可用，記憶體需求高）

  ILSrc = "
    set(preconditioning,ilut,1,0.01);
    set(dropping,1e-4);
  "
}
```

### 4.2 偏壓掃描策略

**IdVg 掃描（關鍵：初始條件要正確）：**

```
Solve {
  # Step 1：先穩定初始條件
  Poisson { }

  # Step 2：加入載子，從平衡態開始
  Coupled { Poisson Electron Hole }

  # Step 3：緩慢升起 Vds（避免跳步）
  Quasistationary(
    InitialStep = 0.01
    MaxStep     = 0.05
    MinStep     = 1e-5
    Goal { Contact="drain" Voltage=0.7 }
  ) { Coupled { Poisson Electron Hole } }

  # Step 4：掃描 Vgs
  Quasistationary(
    InitialStep = 0.01
    MaxStep     = 0.05
    MinStep     = 1e-5
    Goal { Contact="gate_n" Voltage=1.0 }
  ) { Coupled { Poisson Electron Hole } }
}
```

### 4.3 收斂容差調整原則

| 情境 | 建議調整 |
|------|---------|
| 強反型區（高 Vgs）不收斂 | 降低 `MaxStep`，從 0.05V → 0.01V |
| 亞閾值區（低 Ids）震盪 | 提高 `ErRef` 至 1e10，避免低載子濃度數值誤差 |
| 3D CFET 結構整體不收斂 | 先做 2D 截面驗證，再升維 |

## 5. 收斂問題除錯指南

### 5.1 診斷流程

```
不收斂
  │
  ├─ 查看 .log 檔最後幾行
  │    ├─ "Jacobian singular" → 網格問題或模型衝突
  │    ├─ "Newton not converged" → 初始條件或步進過大
  │    └─ "Floating point exception" → 數值溢位（極端摻雜或極薄 EOT）
  │
  ├─ 檢查電流連續性：Jn + Jp 在每個節點是否守恆
  │
  └─ 簡化模型逐一測試（見 5.3）
```

### 5.2 常見不收斂原因與解法

| 症狀 | 根因 | 解法 |
|------|------|------|
| 第一步 Poisson 不收斂 | 摻雜定義錯誤（如 n/p 接面位置異常） | 輸出 doping profile 圖檢查 |
| Vds 升高後發散 | 強電場區網格太粗 | 在 S/D 延伸區加細 |
| 量子模型啟用後不收斂 | DG 與其他模型不相容 | 先關 `QuantumPotential`，收斂後再開 |
| BTB 模型導致發散 | 帶帶穿隧電流數值陡峭 | 加入 `Digits=4`，降低容差要求 |
| CFET 上下層電流不均 | contact 定義錯誤 | 確認上下 gate contact 完全獨立 |
| SiGe 通道發散 | 應變能帶未正確初始化 | 從無應變結構過渡，逐步增加 Ge 成分 |

### 5.3 模型剝離法（逐步排除）

```
1. 只開 Poisson                          → 確認基本靜電正確
2. 加入 Electron + Hole（無量子）        → 確認載子輸運正確
3. 加入 QuantumPotential                 → 確認量子修正不引入問題
4. 加入 Mobility models                  → 確認遷移率模型參數合理
5. 加入 Recombination（含 Band2Band）    → 最後加入複合模型
```

### 5.4 ILS 求解器調參

若使用 ILS 仍不收斂，嘗試：

```
ILSrc = "
  set(preconditioning,ilut,2,0.001);  # 增加 fill-in level
  set(dropping,1e-5);                  # 降低 dropping tolerance
  set(max_iter,200);                   # 增加線性求解器迭代數
"
```

## 6. CFET 特有注意事項

### 6.1 靜電耦合問題

上下層 FET 共用中間本體層時，存在靜電耦合（electrostatic coupling）：
- **症狀**：改變 nFET Vgs 時，pFET Vth 偏移
- **解法**：中間隔離層需足夠厚（> 10 nm SiO₂ equivalent），或在模型中明確定義固定電位邊界

### 6.2 熱效應

CFET 垂直堆疊結構散熱路徑受限：

```
Physics {
  Thermodynamic                  # 啟用熱電耦合模擬
}

Math {
  Coupled { Poisson Electron Hole Temperature }
}
```

- 建議在高電流密度（ON-state）下同時模擬溫度分佈
- 若只做 IdVg，可先關閉 Thermodynamic 以提升收斂性

### 6.3 寄生電阻

CFET 的 contact 路徑比平面 GAAFET 更長：

- 在 SDevice 中明確定義 `SeriesResistance` 或使用 `Contact Resistance` 設定
- S/D epitaxy 的接觸電阻對 Ion 影響可達 15–25%，不可忽略

### 6.4 獨立 Gate Bias 掃描

```
Solve {
  # 固定 pFET gate，掃描 nFET gate
  Quasistationary(
    Goal { Contact="gate_n" Voltage=1.0 }
  ) { Coupled { Poisson Electron Hole } }

  NewCurrentFile = "pFET_sweep"

  # 固定 nFET gate，掃描 pFET gate
  Quasistationary(
    Goal { Contact="gate_p" Voltage=-1.0 }
  ) { Coupled { Poisson Electron Hole } }
}
```

## 7. 常見模擬流程

### 7.1 IdVg 模擬標準流程

```
[SDE] 建立幾何結構 + 定義摻雜 + 網格
  ↓
[SPROCESS] （可選）使用製程模擬替代幾何定義
  ↓
[SDevice] Poisson → 平衡態 → Vds 偏壓 → Vgs 掃描
  ↓
[Inspect / Python] 後處理：萃取 Vth、SS、DIBL、Ion/Ioff
```

### 7.2 關鍵萃取參數

| 參數 | 萃取方法 |
|------|---------|
| Vth | Constant current method（Id = 1e-7 × W/L）|
| SS（次閾值擺幅） | dVgs/d(log Id) 最小值 |
| DIBL | ΔVth(Vds=0.7V) − ΔVth(Vds=0.05V) |
| Ion | Id @ Vgs=Vds=VDD |
| Ioff | Id @ Vgs=0, Vds=VDD |

### 7.3 參數變異模擬建議

對 CFET 進行製程變異分析時，優先考慮：
1. Nanosheet 厚度 ±0.5 nm
2. Gate length ±1 nm
3. Inner spacer 長度 ±1 nm
4. 中間隔離層厚度 ±2 nm
